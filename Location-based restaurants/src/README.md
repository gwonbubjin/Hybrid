# Food Map 🍂

가을 느낌의 위치 기반 맛집 검색 및 여행 코스 생성 앱입니다.

## 🔑 카카오맵 API 설정

이 앱은 카카오맵 API를 사용합니다. 다음 단계를 따라 설정하세요:

### 1. 카카오 개발자 계정 생성 및 앱 등록

1. [Kakao Developers](https://developers.kakao.com/)에서 계정을 만들고 로그인
2. "내 애플리케이션" 메뉴에서 "애플리케이션 추가하기" 클릭
3. 앱 이름과 사업자명을 입력하여 앱 생성

### 2. 플랫폼 설정

1. 생성한 앱 선택 → "플랫폼" 메뉴
2. "Web 플랫폼 추가" 클릭
3. 사이트 도메인 등록:
   - 로컬 개발: `http://localhost:3000`
   - 배포: 실제 도메인 주소

### 3. API 키 발급

1. "앱 키" 탭 선택
2. **JavaScript 키** 복사 (지도 표시용)
3. **REST API 키** 복사 (맛집 검색용)

### 4. 설정 파일에 키 입력

`/config.ts` 파일을 열어 다음 두 개의 키를 입력하세요:

```typescript
export const KAKAO_CONFIG = {
  MAP_API_KEY: "여기에_JavaScript_키_입력",
  REST_API_KEY: "여기에_REST_API_키_입력",
};
```

⚠️ **중요**: API 키 없이는 앱이 정상적으로 작동하지 않습니다.

## ✨ 주요 기능

### 🗺️ 지도 기반 맛집 탐색
- 실시간 위치 기반 주변 맛집 표시
- 카카오맵 연동으로 정확한 위치 정보
- 카테고리별 필터링 (한식, 일식, 중식, 양식, 이탈리안, 카페)
- 거리순, 평점순, 리뷰순 정렬

### ❤️ 즐겨찾기
- 마음에 드는 맛집을 즐겨찾기에 저장
- 로컬 스토리지에 자동 저장
- 언제든지 즐겨찾기 목록 확인 및 관리

### 🚶 하루 보행거리 측정
- 실시간 보행거리 추적
- 일일 목표(10,000걸음) 달성률 표시
- 걸음 수 자동 계산 (1km ≈ 1,250걸음)

### 🗺️ 최적 경로 생성
- 여러 맛집을 담아 최단 경로 자동 생성
- 총 거리 및 예상 소요 시간 계산
- 실시간 경로 안내
  - 현재 목적지 표시
  - 남은 거리 및 시간 표시
  - 진행률 시각화
- 카카오맵 길찾기 연동

### 🎨 가을 테마 디자인
- 따뜻한 가을 색상 (오렌지, 브라운, 베이지)
- 직관적이고 깔끔한 UI/UX
- 반응형 디자인 (데스크톱/모바일)

## 🛠️ 기술 스택

- **프론트엔드**: React, TypeScript
- **스타일링**: Tailwind CSS
- **API**: Kakao Maps API, Kakao Local API
- **아이콘**: Lucide React
- **상태 관리**: React Hooks
- **저장소**: Local Storage

## 📱 사용 방법

### 맛집 검색
1. 검색창에 맛집 이름이나 주소 입력
2. 카테고리 버튼으로 음식 종류별 필터링
3. 정렬 옵션으로 원하는 순서로 정렬

### 즐겨찾기 저장
1. 맛집 카드의 하트 아이콘 클릭
2. 상단 하트 버튼으로 즐겨찾기 목록 확인

### 여행 코스 생성
1. 방문하고 싶은 맛집들을 즐겨찾기에 추가 (2곳 이상)
2. 즐겨찾기 목록에서 "최적 경로로 코스 만들기" 클릭
3. 자동으로 생성된 최적 경로 확인
4. "경로 안내 시작" 버튼으로 네비게이션 시작

## 📋 참고사항

- 위치 권한을 허용하지 않으면 기본 위치(서울시청)가 사용됩니다
- 카카오 로컬 API는 일일 호출 제한이 있을 수 있습니다
- 보행거리는 브라우저 로컬 스토리지에 저장되며, 데이터를 지우면 초기화됩니다
- 평점과 리뷰 수는 카카오 API에서 제공하지 않아 임의로 생성됩니다
