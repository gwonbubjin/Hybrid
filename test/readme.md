# 📌 1. 프로젝트 개요 논의

### ✔️ 개발 조건 확인

* 팀 프로젝트는 **5페이지 이상의 모바일 앱** 제작이 필수
* "휴대폰 하드웨어 기능을 반드시 사용해야 한다"는 말이 있었지만
  → 실제로는 **필수 아님**으로 확인됨
* 따라서 **GPS, 카메라, 센서 등은 선택 사항**, 사용하면 플러스 요소

---

# 💡 2. 제안된 앱 아이디어 목록

## 🔸 1) 날씨 기반 옷 추천 앱

* GPS + 날씨 API 활용
* 카메라로 옷 촬영/저장 기능
* 오늘 날씨 기반 코디 추천

## 🔸 2) 환율 계산 + 카메라 스캔 앱

* 가격표 사진 촬영 → OCR 분석
* 환율 API로 자동 계산

## 🔸 3) 위치 기반 안전귀가 앱

* GPS + Accelerometer(충격/넘어짐 감지)
* 충격 발생 시 등록된 번호로 긴급 메시지 자동 전송
* 이동 경로 지도 표시 기능

## 🔸 4) 위험 소리 감지 알림 앱

* 마이크 입력으로 비명/강한 소음 감지
* 진동 + 경고 화면 표시
* Audio 분석 API 사용

## 🔸 5) 음식 칼로리 스캐너 앱

* 카메라 촬영 → 음식 인식 API 사용
* 칼로리 표시
* 기록은 localStorage 저장

## 🔸 6) “WakeRoute” 지각 방지 알람 앱

* GPS + 지도 API + 알림 + 음성 TTS
* 교통 상황 기반으로 "지금 출발해야 도착 가능" 알림
* 일정 + 경로 기반 알람 시스템

## 🔸 7) 러닝 앱

* 칼로리 계산, 이동 거리 기록
* GPS 기반 트래킹

## 🔸 8) 사물 인식 정보 제공 앱

* 카메라 촬영 → Vision API로 사물 인식
* 검색 API로 정보 요약 제공

## 🔸 9) 위치 기반 맛집 추천 앱 (최종 선정)

* GPS로 현재 위치 기반 주변 맛집/카페 추천
* 지도 API 기반 마커 표시
* 5페이지 이상 구성 가능 → 프로젝트 조건 만족
